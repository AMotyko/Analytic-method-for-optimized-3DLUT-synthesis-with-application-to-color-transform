# Analytic-method-for-optimized-3DLUT-synthesis-with-application-to-color-transform
Analytic method for optimized 3DLUT synthesis with application to color transform

This repository is provided for non-commercial research and educational use only.

This repository contains the reference implementation of an analytical method for optimized 3D lookup table (3D LUT) synthesis for color transformation tasks. The method constructs regular 3D LUTs with soft regularization terms for smoothness, cell-shape regularity, and color-domain preservation, and is intended for device-specific color calibration and RGB–XYZ/RGB–RGB color mapping experiments.

This repository provides research code accompanying the paper “Analytic method for optimized 3DLUT synthesis with application to color transform.” The code is intended for non-commercial research and educational use. It implements analytical optimized 3D LUT construction, regularization-weight selection procedures, and experimental scripts for evaluating color transformation accuracy.
